Spring Boot Subscription Application

What you’ll need to run application: 

About 15 minutes
A favorite Eclipse IDE
JDK 1.8 or later
Maven 3.0+

Steps:
1. Configure project with maven.
2. command to run -> spring-boot:run 
3. input is hardcoded in class and output is generated based on that.

Eg: Input = truncate("123456789012345678901234567890",25)
    Output = Truncated log file: 12 ... (truncated) ... 90
